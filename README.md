## Hey, I'm Parth 👋

I'm a Data Science student from India who genuinely enjoys building things — not just studying them. Whether it's training a model, shipping a web app, or breaking down a complex topic on YouTube, I'm usually somewhere at the intersection of data and code.

A while back, I noticed that viva prep resources for IITM BS students were all over the place. So I built **[VivaVault](https://vivavault.vercel.app/)** — a single platform where students can actually find what they need without the chaos. It's live, it's being used, and that's the kind of stuff that keeps me going.

Currently focused on the AI/ML engineering side of things — model pipelines, deployment, LLM fine-tuning, and whatever else breaks along the way. I also run **[@Datapocalypse](https://www.youtube.com/@Datapocalypse-01)** on YouTube, where I try to make data science feel less intimidating.

- 🔭 Building → AI-powered full-stack apps
- 🌱 Learning → MLOps · LLM Fine-tuning · Model Deployment
- 🤝 Open to → Collabs on DS/ML, Web Dev & student tools
- ⚡ Fun fact → I debug data pipelines like a detective solving cases 🕵️
